
# YouTube Trending Videos Analysis 2026

## 1. Project Overview

YouTube is one of the largest video-sharing platforms in the world, hosting millions of videos in categories like music, gaming, entertainment, education, and news. Every day, millions of users engage with content through views, likes, and comments.

This project analyzes the **YouTube Trending Videos 2026 dataset**, which is a **merged dataset of trending videos from 11 countries**, to understand:

- What types of videos become trending globally.
- Patterns in user engagement (views, likes, comments).
- Country-wise trends and popular channels.
- How trending behavior changes over time.

The insights from this analysis can help content creators, marketers, and researchers understand audience preferences and video performance on a global scale.

## 2. Dataset Information

- **Dataset Name:** YouTube Trending Videos Stats 2026 (Merged 11 Countries)  
- **Source:** YouTube Data API v3  
- **Number of Records:** 178,399+ trending video records  
- **Countries Included:** Brazil, Canada, Germany, France, UK, India, Japan, South Korea, Mexico, Russia, USA  
- **Columns:**  
  - `video_id`: Unique video identifier  
  - `trending_date`: Date video appeared in trending list  
  - `title`: Video title  
  - `channel_title`: Channel name  
  - `views`: Total views  
  - `likes`: Total likes  
  - `dislikes`: Total dislikes  
  - `publish_time`: Original publishing time  
  - `category_id`: Video category  
  - `tags`: Video tags  
  - `comments`: Number of comments  
  - `description`: Video description

## 3. Objectives

1. Analyze **top trending videos globally** by views and engagement.  
2. Explore **popular video categories** and top-performing channels across countries.  
3. Examine **trends across regions** using the merged dataset.  
4. Identify **patterns over time** in trending videos globally.  
5. Prepare dataset for **data visualization and insights generation**.

## 5. Methodology

1. **Data Collection:** Using YouTube Data API v3 for trending videos across 11 countries.  
2. **Data Merging:** Combined all country-specific datasets into **one global dataset**.  
3. **Data Cleaning:** Handling missing values, removing duplicates, and converting date columns.  
4. **Exploratory Data Analysis (EDA):** Understanding views, likes, comments, and category trends globally.  
5. **Visualization:** Plotting top videos, top channels, engagement patterns, and trending over time.  
6. **Insights & Reporting:** Summarizing key findings for both global and country-level analysis.

---

## 6. Key Insights (Summary)

- Music and Gaming videos dominate trending charts globally.  
- Channels with large audiences appear more frequently in trending lists.  
- Videos with higher views usually receive higher likes and comments.  
- Trending activity has increased significantly from 2024 to 2026.  
- USA and India contribute the highest number of trending videos and total views.
